**MovieNight** is a web application that helps manage personal movie watchlist and pick a random movie based on duration filters.  

Built with **JavaScript, HTML, CSS** and using **Firebase** as the backend database.

> **Disclaimer:** This project was created for personal use and as a learning exercise to practice **Firestore Database** tool and apply knowledge of **JavaScript, HTML, and CSS**. It is not intended as a production-ready application.

---

## Features

- Add movies to your watchlist with title and duration  
- Filter movies by length: **short**, **medium**, or **long**  
- Generate a **random movie** from the filtered list  
- Simple, clean, and interactive web interface

---

## Tech Stack

- **Frontend:** JavaScript, HTML, CSS  
- **Backend / Database:** Firebase Realtime Database  

---

## How to Use

1. Open the project in a browser or run a local server.  
2. Add movies to your watchlist.  
3. Use filters to select movie duration.  
4. Click **“Random Movie”** to get a recommendation.
